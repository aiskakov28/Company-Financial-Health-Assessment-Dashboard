## 💹 Project Overview

An interactive web application that analyzes the financial health of publicly traded companies. The dashboard fetches real-time financial data from Finviz and provides comprehensive assessments based on various financial metrics.

## 🎯 Key Features

- Real-time company analysis via ticker symbol input
- Interactive radar and bar chart visualizations
- Comprehensive financial metric breakdowns
- Dynamic exploration of specific metrics

## 🛠️ Technologies

```
Python         - Core programming language
Dash           - Web application framework
Plotly         - Interactive visualization
BeautifulSoup  - Data scraping
HTML/CSS       - Frontend styling
```

## 📁 Project Structure

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/financial-health-dashboard.git
cd financial-health-dashboard
```

1. Install dependencies:

```bash
pip install -r requirements.txt
```

1. Launch the application:

```bash
python main.py
```

1. Access the dashboard:

Open your web browser and navigate to http://127.0.0.1:8050
