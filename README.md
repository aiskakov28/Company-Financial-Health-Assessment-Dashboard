# Company-Financial-Health-Assessment-Dashboard
DISCLAIMER: This project is for educational purposes only. It is not intended for commercial use or to provide financial advice. Users should not rely on this tool for making investment decisions.

**Project Overview:**
This project is an interactive web application that analyzes the financial health of publicly traded companies. It fetches financial data from Finviz and provides a comprehensive assessment based on various financial metrics.

**Features:**
Analyze any publicly traded company by entering its ticker symbol
Visual representation of financial health using radar and bar charts
Detailed breakdown of financial metrics across different categories
Interactive elements allowing users to explore specific metric details

**Technologies Used:**
Python
Dash (for web application framework)
Plotly (for interactive charts)
BeautifulSoup (for web scraping)
HTML/CSS (for styling)

**Installation:**
git clone https://github.com/yourusername/financial-health-dashboard.git
cd financial-health-dashboard
pip install -r requirements.txt
​
**Usage:**
python main.py
Open your web browser and navigate to http://127.0.0.1:8050/

**Project Structure:**
main.py: The main application file containing the Dash app and callbacks
financial_metrics_analysis.py: Contains the logic for financial health assessment
assets/style.css: CSS file for styling the web application
